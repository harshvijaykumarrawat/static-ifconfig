static-ifconfig

CLI tool to manage static IP addresses on Raspberry Pi OS

⬇️ Download prebuilt .deb

Features

  • Set or remove static IP addresses on Raspberry Pi
  • Supports both NetworkManager (modern OS) and dhcpcd (legacy)
  • Simple CLI: -c create, -r remove, -l list
  • Easy Debian package installation

Installation

Download and install the prebuilt package:

wget https://harshvijaykumarrawat.github.io/static-ifconfig/build/static-ifconfig_0.1-1_all.deb
sudo dpkg -i static-ifconfig_0.1-1_all.deb

Or build from source:

git clone https://github.com/harshvijaykumarrawat/static-ifconfig.git
cd static-ifconfig
dpkg -i build/static-ifconfig_0.1-1_all.deb

Usage

Set static IP:

static-ifconfig -c eth0 192.168.1.222/24 192.168.1.1 8.8.8.8

Remove static IP (revert to DHCP):

static-ifconfig -r eth0

List current configuration:

static-ifconfig -l

Compatibility

  • Raspberry Pi Zero / Zero 2 W
  • Raspberry Pi 3 / 3B+
  • Raspberry Pi 4
  • Raspberry Pi 5
  • Tested on Raspberry Pi OS Bookworm and Bullseye
